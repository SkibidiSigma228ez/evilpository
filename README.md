# 🤖 Бот расписания онлайн-школы
## 📋 Возможности

- 📅 Просмотр расписания на неделю
- 🔘 Удобное управление через кнопки
- 📊 Статистика запросов студентов
- 💾 База данных пользователей

## 📁 Структура проекта

```
telegram_schedule_bot/
├── main.py              # Главный файл, запуск бота
├── logic.py             # Логика работы с БД и расписанием
├── schedule_data.py     # Данные расписания (можно редактировать)
├── config.py            # Настройки (токен бота)
├── database.db          # База данных (создаётся автоматически)
├── README.md            # Этот файл
└── requirements.txt     # Зависимости (опционально)
```

## 🗂️ Как обновить расписание

Открой файл `schedule_data.py` и измени данные:

```python
def get_weekly_schedule():
    return {
        "Понедельник": [
            {"time": "10:00", "name": "Математика", "teacher": "Иванова А.А."},
            {"time": "12:00", "name": "Русский язык", "teacher": "Петрова Б.Б."}
        ],
        "Вторник": [
            {"time": "11:00", "name": "Физика", "teacher": "Кузнецов Г.Г."}
        ],
        # ... добавь другие дни
    }
```

Перезапусти бота — изменения применятся.

## 📊 Статистика студентов

База данных `database.db` содержит таблицу `users`:

| Поле | Описание |
|------|----------|
| `user_id` | ID пользователя в Telegram |
| `username` | Имя пользователя |
| `first_request_date` | Дата первого запуска |
| `last_request_date` | Дата последнего запроса расписания |
| `total_requests` | Сколько раз запрашивал расписание |

## 👨‍🎓 Инструкция для студентов

1. Найди бота в Telegram
2. Нажми `/start` или кнопку **Start**
3. Нажми кнопку **📚 Получить расписание**
4. Бот пришлёт расписание на текущую неделю

## 🛠️ Технологии

- Python 3.11+
- aiogram 3.x (асинхронный фреймворк)
- SQLite (лёгкая база данных)
- aiohttp (для работы через прокси)
